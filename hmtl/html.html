¿Qué es HTML?

Introducción

HTML (HyperText Markup Language) es el lenguaje estándar para crear páginas web. Describe la estructura de una página web y sus elementos mediante etiquetas.

Estructura Básica de un Documento HTML

<!DOCTYPE html>
<html>
<head>
    <title>Título de la página</title>
</head>
<body>
    <h1>Encabezado</h1>
    <p>Este es un párrafo.</p>
</body>
</html>

Elementos Comunes

  • <h1> a <h6>: Encabezados
  • <p>: Párrafo
  • <a>: Enlace
  • <img>: Imagen
  • <div>: División o contenedor
