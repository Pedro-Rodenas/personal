¿Qué es PHP?

Introducción

PHP (Hypertext Preprocessor) es un lenguaje de scripting del lado del servidor diseñado para el desarrollo web. Se utiliza para crear contenido dinámico que interactúa con bases de datos.

Estructura Básica de un Documento PHP

<?php
echo "Hola, mundo!";
?>

Conceptos Comunes

  • echo: Imprime texto o variables en la página.
  • $_GET y $_POST: Superglobales para manejar datos de formularios.
  • mysqli y PDO: Extensiones para interactuar con bases de datos MySQL.
  • session_start(): Inicia una sesión para manejar datos de usuario.
  • include y require: Incluir archivos externos en un script PHP.
