¿Qué es CSS?

Introducción

CSS (Cascading Style Sheets) es un lenguaje de diseño gráfico para definir y crear la presentación de un documento estructurado escrito en HTML.

Estructura Básica de un Documento CSS

body {
    font-family: Arial, sans-serif;
    margin: 0;
    padding: 0;
    background-color: #f4f4f4;
}

Propiedades Comunes

  • color: Define el color del texto.
  • font-size: Define el tamaño de la fuente.
  • margin: Define el margen exterior de los elementos.
  • padding: Define el relleno interior de los elementos.
  • background-color: Define el color de fondo de los elementos.
